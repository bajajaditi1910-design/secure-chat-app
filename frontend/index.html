About the Project

This secure chat application implements military-grade end-to-end encryption using Python sockets and the cryptography library. The system ensures complete message privacy through a blind relay server architecture where the server cannot decrypt or read any messages.

Built with a focus on security and privacy, the application demonstrates fundamental cryptographic principles and secure communication protocols in a practical, real-world implementation.

Security Features

Industry-standard cryptographic protocols

Diffie-Hellman Key Exchange

Secure key establishment protocol that allows two parties to establish a shared secret over an insecure channel.

AES Encryption

Advanced Encryption Standard (AES) in CBC mode. All messages are encrypted with 256-bit keys for military-grade protection.

HMAC Integrity

Hash-based Message Authentication Code (HMAC) ensures message integrity and authenticity, protecting against tampering.

Blind Relay Server

The relay server has zero knowledge of message contents. All encryption and decryption happens on client devices.
