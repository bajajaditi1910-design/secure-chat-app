Secure Chat Application

A Python-based secure messaging system implementing Diffie-Hellman key exchange, AES encryption, HMAC integrity checks, and a zero-trust relay server.

Security Features

Industry-standard cryptographic protections

Diffie-Hellman Key Exchange

Secure session key establishment with perfect forward secrecy.

AES-256 Encryption

All messages encrypted using AES in CBC mode.

HMAC Integrity

SHA-256 HMAC verification to prevent tampering.

Zero-Trust Server

Server acts only as a blind relay with no access to plaintext.
